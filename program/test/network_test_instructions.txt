匈汉象棋网络对战功能测试说明

1. 单机测试方法：
   a. 打开第一个命令行窗口，运行：
      python server_only.py
      
   b. 打开第二个命令行窗口，运行：
      python client_only.py
   
   c. 两个实例将通过本地回环地址(127.0.0.1)进行通信

2. 正常运行流程：
   - 服务器实例将显示等待连接的状态
   - 客户端实例将尝试连接到服务器
   - 连接成功后，双方可以开始对战

3. 故障排除：
   - 如果遇到端口占用问题，请检查是否有其他实例在运行
   - 确保防火墙没有阻止Python进程的网络访问
   - 如果连接失败，检查网络配置和端口设置

4. 代码文件说明：
   - xhlan.py: 网络通信核心功能
   - network_game.py: 网络对战游戏逻辑
   - network_constants.py: 网络配置常量
   - server_only.py: 仅服务器模式
   - client_only.py: 仅客户端模式
   - test_local_network.py: 单机测试脚本

5. 注意事项：
   - 服务器必须先启动，客户端后连接
   - 游戏中的网络功能目前还在开发阶段
   - 在正式发布前可能需要进一步调试和完善
